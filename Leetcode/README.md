# Leetcode

LeetCode是一個收集軟體工程師面試考古題的線上練功網站。一般課程都是教你怎麼樣快速做出有成就感的東西，主要在「應用」層面，比如說做出幾款應用等等、表格地圖資料庫範圍。而Leetcode的題目比較偏向資料結構和演算法，這些東西就屬於學科裡的硬底子功夫，像是Binary Tree和Linked List，所以刷Leetcode較能增強我們的基本功。
